ASR Dress Makers is a premium tailoring and boutique business specializing in:

🧒 Kids Wear

This project transforms the business from relying solely on social media into a professional digital storefront where customers can browse products, explore collections, and interact with the brand more efficiently.

🧠 Challenges Solved

Dynamic Product Rendering

Created reusable rendering functions for scalable product management.


---

Shopping Cart Persistence

Implemented Browser LocalStorage to preserve user cart data after page refresh.


---

Product Navigation Bug

Identified and fixed a CSS ::after overlay that blocked click events using Chrome DevTools and pointer-events.


---

Product Data Management

Separated predefined products from dynamically added products while maintaining consistent rendering.
